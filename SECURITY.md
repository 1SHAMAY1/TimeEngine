# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within TimeEngine, please follow these steps:

1. **Do not** disclose the vulnerability publicly (e.g., via GitHub Issues).
2. Email the maintainer at [insert email or use private security advisory on GitHub].
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.
